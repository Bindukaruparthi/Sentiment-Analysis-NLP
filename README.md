# Sentiment-Analysis-NLP
Sentiment Analysis using NLP Pipeline and ML Models

## Project Overview
This project performs sentiment analysis using NLP techniques and machine learning models.

## Steps Covered
- Data preprocessing (cleaning, stopwords removal, stemming)
- Feature engineering (TF-IDF, Bag of Words)
- Model training (Logistic Regression, Naive Bayes, Decision Tree)
- Model evaluation (Accuracy, Precision, Recall, F1 Score)

## Result
Logistic Regression performed best with TF-IDF features.
